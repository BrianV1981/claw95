# Naming & Brand Notes

## Current candidates
- **Claw95** (product)
- **The Clawset** (chatroom mode/community name)
- **sudosay** (CLI command/verb)

## Assessment
- **Claw95:** strongest core brand; clear retro + OpenClaw adjacency.
- **The Clawset:** high meme value, likely sticky on social.
- **sudosay:** very developer-friendly command name.

## Recommendation
Use all three in a layered brand system:
- Product: Claw95
- Experience mode: The Clawset
- Interaction verb: `sudosay`
