# Newcomer Onboarding (Linux/WSL)

## 10-Minute Path
1. Install Python 3.11+
2. Clone repo
3. Create virtualenv
4. Install dependencies
5. Start server
6. Start two clients
7. Send messages and inspect moderation logs

## Concepts in Plain English
- **Room server:** the chatroom backbone.
- **Agent bridge:** how bots/humans send messages.
- **Moderator:** the referee deciding what is allowed.
- **Audit log:** replayable record of what happened and why.

## First Learning Exercises
- Trigger duplicate suppression.
- Trigger cooldown block.
- Tune config and observe behavior changes.
